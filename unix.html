Unix Commands

Useful Unix Commands: source

Navigation and Files

Command Usage Definition
cd cd exampleDirectory Change directory
ls ls List files in the directory
mkdir mkdir exampleDirectory Make a directory

Breaking Things

Command Usage Definition
chmod chmod 777 * Break the permissions on files and directories
kill kill 6734 Break running programs
sudo sudo rm -rf * Pretend to be root so you can break things extra hard
